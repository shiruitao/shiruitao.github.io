First-Class Functions in Go

规则：两个玩家掷骰子，每回合你可以选择掷或不掷。

  • 如果掷到点数1，这回合得到的点数全部清空，并切换玩家。
  • 如果玩家选择停止此回合，此回合得到的点数加到该玩家的总点数（之前的点数和 + 当前回合点数），然后切换玩家

先到100点的玩家获胜。

score 结构体包括：当前玩家点数，对手点数，本回合累积获得的点数
